# Scrimble Tech Stack

## Core Language
- **TypeScript (Node.js)**: Ensuring robust type safety and modern JavaScript features across the entire monorepo.

## Frameworks & Libraries
- **oclif**: Providing a robust, extensible CLI framework with command routing and plugin support.
- **Ink (React)**: Enabling a declarative, React-based approach to building the CLI's interactive user interface components.
- **Vercel AI SDK**: Powering multi-provider AI model integration (OpenAI, Anthropic, Google, etc.) with a unified interface.
- **Zod**: Implementing strict schema validation for configuration and data structures.
- **Chalk**: Enhancing the CLI output with formatted terminal colors for improved readability.

## Tooling & Architecture
- **Turborepo**: Managing the monorepo structure and optimizing build, lint, and test tasks.
- **pnpm**: Managing workspace dependencies efficiently with a shared lockfile.
- **Vitest**: Conducting unit and integration tests with a high-performance test runner.
- **ESLint**: Enforcing consistent coding standards and identifying potential issues.
